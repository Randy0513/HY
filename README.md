# House.Yo 好室優

🏠 一站式智慧租屋平台，提供房東與租客最流暢的媒合體驗。

---

## 🔗 網站連結
👉 [https://randy0513.github.io/HY/House.Yo/login.html](https://randy0513.github.io/HY/House.Yo/login.html)

---

## 🚀 功能特色
- ✅ 手機號碼 + 密碼登入（Email 模擬登入機制）
- ✅ 支援 Google 第三方登入
- ✅ 註冊 / 登入 / 登出 完整流程
- ✅ RWD 響應式版型，支援手機、平板、桌機
- ✅ 使用 Firebase Authentication 實作安全認證
